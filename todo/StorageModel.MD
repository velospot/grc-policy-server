14. Storage strategy

Use polyglot persistence.

Relational DB

For:

documents
versions
users
permissions
workflows
job states
Document store / object storage

For:

original uploads
parsed JSON
OCR artifacts
images
table snapshots
generated reports
Graph DB or graph layer

For:

document hierarchy
clause relationships
version mappings
cross-document semantic links
control mappings
Vector DB

For:

multilingual semantic retrieval
clause similarity
chat retrieval
Search index

For:

lexical search
filters by standard, revision, industry, language, jurisdiction
