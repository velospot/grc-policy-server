# UML Diagram Sources

These are Mermaid source files for architecture and workflow visualizations.

## Files

- `system-context.mmd`: context diagram (user, UI, API, dependencies).
- `component.mmd`: component-level architecture.
- `upload-sequence.mmd`: asynchronous upload sequence.
- `compare-sequence.mmd`: asynchronous compare sequence with cache branch.
- `deployment.mmd`: deployment-level topology for scale-out.
- `domain-class.mmd`: core API/domain class model.
- `job-state.mmd`: job lifecycle state machine.

## Usage

Copy file content into any Mermaid-compatible markdown renderer.
