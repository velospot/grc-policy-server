# Document Management System Docs

This folder contains presentation and architecture deliverables for the document management and processing system based on:

- Backend: `/Users/navm/projects/grc-policy-server`
- Frontend: `/Users/navm/projects/ectest-audit-v2`

## Contents

- `presentation.md`: 15-minute presentation deck (markdown slides).
- `architecture.md`: architecture overview (current state + target state).
- `design.md`: detailed system design and decisions.
- `workflow.md`: end-to-end operational workflows.
- `uml/`: standalone UML diagrams in Mermaid format.

## UML Files

- `uml/system-context.mmd`
- `uml/component.mmd`
- `uml/upload-sequence.mmd`
- `uml/compare-sequence.mmd`
- `uml/deployment.mmd`
- `uml/domain-class.mmd`
- `uml/job-state.mmd`

## How To Use

1. Use `presentation.md` directly for a short presentation.
2. Embed or paste diagrams from `uml/*.mmd` into markdown renderers that support Mermaid.
3. Use `architecture.md`, `design.md`, and `workflow.md` as speaker notes or submission documents.
